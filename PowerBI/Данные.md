# КАКИЕ ОТЧЕТЫ ТРЕБУЕТСЯ ПОЛУЧИТЬ
НДС равен 18%
1. Общий отчет по участникам программы.
   
   ### Необходимо выводить показатели:
  - Кол-во уникальных покупателей;
  - Кол-во покупок (транзакций);
  - Средний чек, без НДС (т.е. сколько рублей приносит одна транзакция);
  - Выручки, без НДС.

    
      ### Необходима возможность фильтровать показатели:
  - По городам проживания участников программы;
  - По периодам (месяц-год) вступления участника в программу (т.е., например, все
    декабрьские 2017 года, майские 2018 и т.д.).
